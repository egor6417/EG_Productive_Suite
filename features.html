    EG Productive Suite

    Need For Work

    EG Productive Suite is text editor, a word counter, a checklist, a pomodoro timer and a clock. Let's check out each one in detail:

    1. Text editor is a simple, no-distractions text box

    2. Word counter, well, counts words

    3. Checklist is a simple to-do list, also has a little notepad

    4. Timer is a pomodoro timer

    5. The clock shows the time
